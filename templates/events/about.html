About the website

College Events website is a web-application which displays the current departmental, club and community events happening at Vasavi College of Engineering. The application is responsive, robust and reliable and is developed to overcome the obfuscation in the current college website and enhance the experience of its users.


The project is divided in three applications - accounts, events and forum. The accounts application handles all the functions related to user login, signup, password validation, sessions handling, email confirmation, etc. This is provided by allauth library, which is installed as an application in this project. The events application deals with the events. This application allows authenticated users to create, view, update and delte an event. Every event is associated with an organiser. The user has an option to get notification to their email whenever there is an update for an event he chooses. The forum app is a section where users can ask questions and give answers to an event. Only verified users are allowed to post questions and answers.


The back-end of the web-application is developed using Django2.0 framework and few utility functions were written in Python3. The database used for the project is SQLite, which is the default database for Django. The front-end of the web-application is developed using Bootstrap4, HTML5 and CSS3.
